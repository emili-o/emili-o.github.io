# Not all file systems have case sensitivity.

One of the reasons for naming files without spaces or special characters is that not all of them are case-sensitive.[^1]

Windows and Mac are both case-insensitive.

[^1]: For readability, I do not adhere to these rules in my Obsidian vault.
