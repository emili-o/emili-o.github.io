# [[HTML]]

## Document Object Model

The **DOM** refers to the tree-like structure of an HTML file.

The model helps us to understand which HTML elements are contained within other elements.

![From CS50p](https://i.imgur.com/Os5TqzA.png)

⬆️ The DOM for the following HTML document[^1]:

```html
<!doctype html>
<html lang="en">
  <head>
    <title>Hello!</title>
  </head>
  <body>
    Hello, world!
  </body>
  <html></html>
</html>
```

[^1]: figure from [Lecture 0 - CS50's Web Programming with Python and JavaScript](https://cs50.harvard.edu/web/2020/notes/0/)
